Question: Difference between local storage and session storage.

Ans: 1. The data in localStorage is permanent and the data in sessionStorage is temporary
2.If we want to add data to localStorage write rules - localStorage.setItem ('key', value) and If we want to add data to sessionStorage write rules - sessionStorage.setItem ('key', value)
3.If we want to remove data to localStorage write rules - localStorage.removeItem ('key') and If we want to remove data to sessionStorage write rules - sessionStorage.removeItem ('key')
4.If we want to get data to localStorage write rules - localStorage.getItem ('key') and If we want to get data to sessionStorage write rules - sessionStorage.getItem ('key')
5.If we want to clear data to localStorage write rules - localStorage.clear () and If we want to clear data to sessionStorage write rules - sessionStorage.clear ()

Question: What is javaScript event loop do?

Ans: JavaScript is a single-threaded language, which means it has only one call stack that is used to execute the program. Some parts of JavaScript run synchronously and some parts run asynchronus. When javascript works, it has two parts, stack and Queue. After completing the work of the frames of the stack, Queue's messages run one by one. javascript This process is called event loop.
